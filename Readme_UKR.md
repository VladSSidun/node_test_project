# CodeCare — Basic Coding Skill Challenge

## Завдання

Читати stdin рядок за рядком і виводити тільки ті рядки, в яких **хоч один символ зустрічається рівно 2 рази**.

---

## Алгоритм

1. Для кожного рядка підраховуємо частоту кожного символу (`freq` map).
2. Якщо хоча б одне значення в `freq` дорівнює `2` — виводимо рядок.
3. Рядки, де жоден символ не зустрічається рівно двічі — пропускаємо.

**Складність:** O(n × k), де n — кількість рядків, k — довжина рядка.

---

## Запуск

### Варіант 1 — через pipe з файлу

```bash
node solution.js < input.txt
```

### Варіант 2 — інтерактивно (вводити руками, Ctrl+D для завершення)

```bash
node solution.js
```

### Варіант 3 — через npm

```bash
npm start < input.txt
```

---

## Тест (еталон із завдання)

```bash
npm test
# або
node test.js
```

### Sample Input

```
asdf
fdas
asds
d fm
dfaa
aaaa
aabb
aaabb
```

### Expected Output

```
asds
dfaa
aabb
aaabb
```

### Пояснення

| Рядок | Символи з count=2 | Результат |
| ----- | ----------------- | --------- |
| asdf  | немає             | ❌        |
| fdas  | немає             | ❌        |
| asds  | s (×2)            | ✅        |
| d fm  | немає             | ❌        |
| dfaa  | a (×2)            | ✅        |
| aaaa  | немає (a=4)       | ❌        |
| aabb  | a(×2), b(×2)      | ✅        |
| aaabb | b (×2)            | ✅        |
